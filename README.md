# 💎 LUMIÈRE — Luxury Jewelry Website

موقع لوميير للمجوهرات الفاخرة | A Beautiful Arabic Luxury Jewelry E-Commerce Website

![LUMIÈRE](https://img.shields.io/badge/LUMIÈRE-Luxury%20Designs-gold?style=for-the-badge)

## 📋 نبذة عن المشروع

**LUMIÈRE** هو موقع فاخر متخصص في بيع المجوهرات الفاخرة والتصاميم الحصرية. يجمع الموقع بين التصميم الأنيق والواجهة الاحترافية مع دعم كامل للغة العربية.

## ✨ المميزات الرئيسية

- 🎨 **تصميم فاخر وأنيق** - واجهة مستخدم راقية تعكس الفخامة
- 📱 **متجاوب بالكامل** - يعمل بشكل مثالي على جميع الأجهزة
- 🌐 **دعم اللغة العربية** - واجهة عربية متكاملة
- 🔔 **نوافذ منبثقة تفاعلية** - عرض تفاصيل المنتجات
- 💳 **خيارات دفع متعددة** - نقدي، Apple Pay، تابي
- 📦 **نموذج طلب تصميم خاص** - للعملاء الراغبين بتصاميم فريدة
- 🎯 **أيقونات جميلة ورسومات احترافية**

## 🛠️ التخصيص

### تغيير معلومات الاتصال
عدّل رقم الواتس في دالة `openWhatsApp()`:
```javascript
function openWhatsApp() {
  const message = encodeURIComponent('رسالتك هنا');
  window.open(`https://wa.me/966XXXXXXXXX?text=${message}`, '_blank');
}
```

### إضافة منتجات جديدة
أضف كارت جديد في قسم `#products`:
```html
<div class="card-p" onclick="openModal(event)">
    <img class="ph" src="صورة_المنتج.jpg" alt="Product">
    <div class="body">
        <div class="name">اسم المنتج</div>
        <div class="meta">وصف المنتج</div>
        <div class="from">ابتداءً من <b>السعر</b> ر.س</div>
    </div>
</div>
```
